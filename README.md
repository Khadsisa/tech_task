СКРИПТ ДЛЯ ПРОВЕРКИ E-MAIL ДОМЕНОВ
  - e-domen_checker

1.Установите нужные модули для для преобразования доменных имен и для параллельного выполнения задач: pip install dnspython, concurrent.futures
2.Импорт модуля в файле, корневой директории
3.Создание функции для MX-записей домена и вывода статуса адреса
4.Создание списка доменов с проверкой на корректность ввода.
Конструкция с with для параллельной обработки и кэширования результатов - ускорения процесса
5.Запуск приложения

МИНИ ИНТЕГРАЦИЯ С TELEGRAM(принимает текст из файла .txt и отправляет этот текст в выбранный приватный Telegram-чат через Telegram-бота)
 - tg_tz

Для создания:
1.Установите: pip install requests
2.Создайте переменные- токен бота в config.py, айди чата и путь к файлу: FROM CONFIG IMPORT TOKEN; CHAT_ID, FILE_PATH
3.Далее следует создание функции по отправке текста при помощи response
4.В функции main объедините чтение файла txt с отправкой в чат, при помощи ранее созданной функции
5.Запустите приложение
